CSS

LOCALIZACIÓN

Para localizar de forma relativa un elemento se usa "div.relative {position: relative; left: 10px;"

Para mostrar un elemento se usa li {display: inline;}

Para establecer los margenes se usa "p {margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 150px;}". Top = Margen desde arriba - Left = Izquierda - Right = Derecha. También puedes usar "margin: 150px;" y todos tus márgenes se estableceran a 150px.

El Padding funciona de manera similar al Margin, pero en este caso hace un espaciado alrededor de un elemento. "div {padding-top: 50px; padding-right: 30px; padding-bottom: 50px; padding-left: 80px;}"

Como su propio nombre indica, el text-align cumple la función de alinear el texto de manera similar a como lo hacen los procesadores de texto "div.a {text-align: x;}". x puede ser remplazada por "center", "left", "right" o "justify"

Para escoger un color hay tres maneras principales de hacerlo: eligiendo uno de los colores prestablecidos ("red", "blue", "yellow"...), por codigo de color ("#00ff00", "#435E44"...) o por código RGB ("rgb(0,50,255)"). body {color: red;}

Background cambia ajustes del fondo de la páginas tales como el color. "div {background-color: lightblue;}"

Border establece un borde alrededor de un elemento, para ello se usa "p.solid {border-style: solid;}". Para diferentes estilos remplaza "solid" (solido) por "dotted" (punteado), "dashed" (guiones)...

Para cambiar la fuente de un texto usa p {font-family: "Times New Roman", Times, serif;}, remplazando Times New Roman", Times, serif; por la fuente que desees usar.

Box-Shadow "{box-shadow: 5px 10px #888888;}".
